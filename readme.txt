== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `xBooster Social Plugin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Thats all, you can customize the plugin by opening the Settings > xBooster Social Plugin
